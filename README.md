# 🧠 AI-Based Stress Detection & Measuring System

An intelligent machine learning-based system that predicts stress percentage, identifies root causes, and provides AI-powered recommendations based on daily lifestyle parameters.

---

## 🚀 Problem Statement

Modern students and professionals experience rising stress levels due to academic pressure, work overload, financial concerns, social isolation, and lifestyle imbalance.

However, most stress detection systems:
- Do not quantify stress accurately
- Do not explain the causes
- Do not provide actionable recommendations

This project solves that gap.

---

## 💡 Solution Overview

The AI Stress Detection System:

✔ Predicts Stress Percentage (0–100%)  
✔ Classifies Stress Level (Low / Moderate / High)  
✔ Identifies Root Causes from 11 parameters  
✔ Provides Personalized AI Suggestions  
✔ Displays 24-Hour Stress Projection  

Built using Machine Learning and deployed with Streamlit.

---

## 📊 Input Parameters (11 Features)

1. Sleep Hours  
2. Work Hours  
3. Social Interaction  
4. Physical Activity  
5. Mood Level  
6. Caffeine Intake  
7. Screen Time  
8. Financial Pressure  
9. Work Pressure  
10. Relationship Stress  
11. Anxiety Level  

---

## 🤖 Machine Learning Model

- Algorithm: Random Forest Regressor  
- Target: Stress Percentage (0–100)  
- Dataset: Synthetic structured stress dataset  
- Features: 11 lifestyle & psychological indicators  
- Evaluation: Regression-based stress prediction

## ▶ How to Run Locally

1. Clone the repository
2. Install dependencies:

```

pip install -r requirements.txt

```

3. Run the app:

```

streamlit run app.py

```
